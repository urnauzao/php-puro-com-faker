# Sobre
Projeto com um exemplo simples de inicialização de projeto PHP sem framework usando o FakerPHP.

# FakerPHP
[Acessar FakerPHP](https://fakerphp.github.io)

# Requerimentos
- Ter PHP 7.1 ou superior instalado no ambiente.
- Ter o Composer instalado no ambiente.

# Instalação
- Clone o projeto
- Para instalar as dependências do Composer execute:
> composer install

- Para iniciar o PHP você pode executar no terminal o comando:
> sh start_server.sh

- Ou, se preferir rode no terminal o comando:
> php -S localhost:8000

Você pode alterar a porta 8000 que está no comando acima para qualquer porta que estiver disponível em sem ambiente.
